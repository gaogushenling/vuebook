1.1. Vue简介

Vue.js的作者为Evan You（尤雨溪），任职于Google Creative Lab。

Vue的主要特点就和它官网（http://cn.vuejs.org/）所介绍的那样：

（1） 简洁 （2） 轻量 （3）快速 （4） 数据驱动 （5） 模块友好 （6） 组件化

1.2. Vue.js的快速入门 Hello World

1.2.1. 安装Vue环境

Vue.js 官网下载地址：http://vuejs.org/guide/installation.html，推荐使用开发版本，因为里边会由很多的错误提示等等，

也可以使用在线的文件：

<script src="https://cdn.jsdelivr.net/npm/vue@2.5.17/dist/vue.js"></script>

1.2.2. 开始使用Vue来实例化对象赋值

<!DOCTYPE html>
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <meta charset="utf-8" />

    <title>hello Vue.js</title>
    <!--引用vue.js在线文件，安装-->
    <script src="https://cdn.jsdelivr.net/npm/vue@2.5.17/dist/vue.js"></script>
</head>
<body>
    <!--定义一个容器 id="app"-->
    <div id="app">
        hello：{{ name }} <!--获取变量 name Mustache语法 两个花括号 -->
    </div>

    <script>
        //定义全局变量vm
        let vm = new Vue({
            el: "#app",
            data: {
                name: "vue.js",
            }
        });
    </script>
</body>
</html>

1.2.3. 数据控制页面

我们知道，vm就是我们定义的全局变量，如果我们想修改其中的值，我们可以这样

    <script>
        //定义全局变量vm
        var vm = new Vue({
            el: "#app",
            data: {
                name: "vue.js",
            }
        });

      vm.name="Hello Blog"
      //除了数据属性，Vue 实例还暴露了一些有用的实例属性与方法。它们都有前缀 $，以便与用户定义的属性区分开来。
      vm.$data.name="Hello Blog3"
      vm._data.name="Hello Blog8"
    </script>

1.3. Chrome插件-vue.js-devTools安装和使用

由于Vue数据驱动的特点，在开发调试中查看DOM结构不是很方便，在Chrome浏览器下调试Vue时,使用Chrome-vue.js-devTools插件,对数据结构进行解析和调试。

安装:

将插件安装文件拖拽到Chrome扩展程序窗口下即可

使用:

安装完成后,关闭浏览器重新打开,加载Vue页面,F12进入Chrome调试,会出现新的Vue-Tab标签
